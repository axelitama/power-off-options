# Power Off Options

Power Off Options은 컴퓨터 끄기 대화창에 아래 버튼을 추가하는 GNOME Shell 확장입니다.

- **화면 끄기** — 즉시 화면을 끕니다.
- **하이브리드 절전** — 메모리를 사용한 절전과 동시에 디스크에 시스템 상태를 저장합니다. (전원 제거 대응)
- **절전 후 최대 절전** — 절전 후 일정 시간이 지나고 자동으로 최대 절전을 합니다.
- **최대 절전** — 시스템 상태를 디스크에 저장한 후 전원을 종료합니다. (절전보다 느림)
- **사용자 공간 다시 시작 (소프트 리부트)** — 시스템 전체를 다시 시작하지 않고 사용자 공간만 다시 시작합니다.
- **BIOS로 다시 시작** — 다시 시작 후 시스템 펌웨어 및 BIOS 설정으로 진입합니다.
- **사용자 정의 명령** — 설정 창에서 사용자 정의 명령을 추가할 수 있습니다. 사용자 정의 명령을 사용하여 실행하고자 하는 임의의 쉘 명령어를 전원 메뉴에 추가할 수 있습니다.

## 요구사항

- GNOME 쉘;
- 화면 끄기 명령은 X11에서만 작동합니다. Wayland는 지원하지 않습니다;
- 최대 절전, 하이브리드 절전 및 다시 시작 관련 명령은 각각에 해당하는 `systemctl` 명령을 사용하며 관련 기능이 활성화되어 있어야 작동합니다.

## 설치

GNOME 확장 웹페이지에서 설치합니다:  
<https://extensions.gnome.org/extension/8189/power-off-options/> 

또는 수동으로 설치할 수 있습니다:

1. 이 저장소를 내려받거나 복제합니다;
2. 확장 디렉토리로 이동합니다;
3. 아래 명령어를 실행합니다:
    - `make` 확장 설치;
    - `make uninstall` 확장 제거;
4. GNOME 쉘을 다시 시작하여 변경 사항을 적용합니다. (로그아웃 후 재 로그인 등)

## 설정

포함된 설정 창에서 각 버튼을 개별적으로 활성화하거나 비활성화할 수 있습니다:

```bash
gnome-extensions prefs power-off-options@axelitama.github.io
```

**GNOME 확장** 앱에서도 설정 창에 접근할 수 있습니다.
